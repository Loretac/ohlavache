# Oh La Vache!

Help La Vache protect his city from the aliens. Move with the arrow keys and shoot with the spacebar. 

See a demo [here.](https://www.youtube.com/watch?v=NpaKdOW1ymc)

## Built With

* [Qt](https://www.qt.io)
* C++
* Adobe Photoshop
* Adobe Illustrator
