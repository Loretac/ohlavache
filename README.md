# Oh La Vache!

Help La Vache protect his city from the evil aliens. Move with the arrow keys and shoot with the spacebar. 

See a demo [here.](https://www.youtube.com/watch?v=NpaKdOW1ymc)

* C++
* Adobe Photoshop
